Drag.js Paths & Configuration - QA Test Suite

Configuration Reference

Attribute: data-tauri-drag-region

Drag Region Values

bare / no value → self (direct clicks only)
"deep" → deep (subtree clicks)
"false" → disabled (drag blocked)
